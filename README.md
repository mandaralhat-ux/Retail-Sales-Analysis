Retail Sales Analysis using the Superstore Dataset
Project Overview
This project was completed as part of my Data Analyst Internship at LA Green Solutions during June–July 2026.
The objective of the project was to analyze retail sales data using SQL and Microsoft Excel to identify business trends, evaluate sales performance, and generate actionable business insights.

Objectives
	•	Analyze retail sales performance.
	•	Evaluate profit across different product categories.
	•	Study regional sales trends.
	•	Identify top-performing states.
	•	Create an interactive dashboard for business decision-making.
	•	Present findings through professional documentation.

Tools Used
	•	Microsoft Excel Online
	•	MySQL Workbench

Dataset
The project uses the Superstore dataset containing information such as:
	•	Orders
	•	Sales
	•	Profit
	•	Discount
	•	Quantity
	•	Category
	•	Sub-Category
	•	Region
	•	State
	•	Customer Segment
	•	Shipping Details

Project Workflow
	•	Data Exploration
	•	SQL Analysis
	•	KPI Verification
	•	Dashboard Creation
	•	Business Insights
	•	Executive Summary
	•	Business Recommendations
	•	Final Project Documentation

Dashboard Features
	•	Sales by Category
	•	Profit by Category
	•	Sales by Region
	•	Profit by Region
	•	Top 10 States by Sales
	•	Key Performance Indicators (KPIs)

Key Insights
	•	Technology generated the highest sales.
	•	Profitability varied across regions.
	•	A small number of states contributed significantly to overall sales.
	•	Regional analysis highlighted opportunities for business expansion.

Project Outcome
The project successfully transformed raw retail sales data into meaningful business insights through SQL analysis, Excel dashboards, and professional documentation. The final deliverables support informed business decision-making and demonstrate practical data analytics skills.
